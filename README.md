# kicad-my-symbols

My KiCad schematic symbol library.

## Installation

```
git clone https://github.com/sebmalissard/kicad-my-symbols ${HOME}/ws/kicad/kicad-my-symbols
```

## Configuration

Add the library in KiCad Eeschema, go to "Preferences" > "Manage Symbol Libraries..."

Then set the library path "${HOME}/ws/kicad/kicad-my-symbols/sym-lib-table"

To finish set library format **Table**.
